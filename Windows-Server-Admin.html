Use Active Directory Users and Computers (ADUC) to create and manage user accounts. Navigate to Start > Administrative Tools > Active Directory User and Computers, and manage user properties such as user password, group membership, and security settings.

In the Server Manager, go to roles and features, select the DNS Server role, and configure DNS zones (forward and reverse lookup). You can manage DNS settings thorough the DNS Manager by setting up A, MX, CNAME, and other record types.

Open Server Manager, go to Add Roles and Features, Select the DHCP Server role, and configure scopes, leases, and IP address ranges. Once installed, use the DHCP Manager to configure IP distribution settings.

Use Windows Server Update Services (WSUS) to manage updates centrally. You can configure automatic updates through Group Policy or manually approve and deploy updates via WSUS to specific servers or group of devices.

Windows Server Backup, a built-in feature, allows you to schedule regular backups including full server of individual file-level backups. Third party backup solutions (e.g., Veeam, Acronis) are often used for advanced features like offsite backups and disaster recovery.
